코덱스 텔레그램 브리지

텔레그램 봇, 작업 폴더, Codex CLI 연결을 이 화면 한 곳에서 설정합니다.

1. 봇 토큰 입력
2. 내 Telegram ID 입력
3. 작업 폴더 지정
4. 저장 후 봇 테스트

브리지 설정

왼쪽만 채워도 기본 동작은 됩니다. 막히면 오른쪽 진단에서 Codex 설치와 로그인 상태를 먼저 보세요.

빠른 안내 `Allowed User IDs`에는 내 텔레그램 숫자 ID를 넣고, `Workspace Path`에는 Codex가 작업할 폴더를 넣으면 됩니다.
텔레그램에서 보거나 구분할 이름입니다.
`@BotFather`에서 받은 긴 문자열을 그대로 붙여넣으세요.
내 텔레그램 숫자 ID입니다. 여러 명이면 쉼표로 구분합니다.
Codex가 파일을 읽고 작업할 실제 폴더입니다.
보통 비워둬도 됩니다. 자동 탐지가 안 될 때만 `codex.cmd` 또는 `codex.exe` 경로를 넣으세요.
봇의 기본 말투와 작업 스타일입니다.
기본값은 600초입니다.
현재 입력한 토큰을 잠깐 확인할 때만 켜세요.
